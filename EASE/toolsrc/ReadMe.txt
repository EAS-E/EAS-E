These are the EAS-E project sources for the EAS-E system.

NEWTRAN is the source project for EASEtranslate
 - the generated .cpp/.h files are placed in gennedX folder
 - these files should be copied to the toolsdev project for the translator

ESETS is the source for easeSets.cpp/.h which are in the lib folder

ESERVER is the source for the EAS-E components of the persistence server.
